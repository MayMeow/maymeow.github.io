Content management with CakePHP

Today I want to show you how easily you can create simple content management system with CakePHP. I want the post can have child posts so we use Tree component from cakePHP.

What we will need?

PHP CakePHP Compsoer

Setup application

Download CakePHP application, you can use guide on Cakephp page, but I recomend you to use jdmaymeow/app repo. This repository is updated cakephp/app so you dont need edit app.controller. This repoisitory i using in all Code Advent 2016. So lets create our app:

composer create-project --prefer-dist jdmaymeow/app content-management

Next we will need User authentication, what we created in previous post. So intall it with composer

composer require jdmaymeow/cake-auth
composer require jdmaymeow/cake-utility
composer require jdmaymeow/cake-taxonomy

If you following tutorial you don’t need to install items above.

Don’t forget to load plugins with bin/cake plugin load. Now we have setup all what we need, and can create content management plugin.

Content management Setup

Create plugin

bin/cake bake plugin CakeContent -f

This will create your new plugin and setup bootstrap for you.

Create Tables

bin/cake bake migration CreateNodes title:string body:text user_id:integer category_id:integer active:boolean published:boolean parent_id:integer lft:integer rght:integer created modified -p CakeContent
bin/cake migrations migrate -p CakeContent

Tables are created and now you can create models, controllers and views. bin/cake bake all Nodes -p CakeContent
