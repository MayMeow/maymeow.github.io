Taxonomy with CakePHP

Hello. In this part i watn to show you how to create simple taxonomy manager to categorize all your content on webpage. It will have 3 parts. Terms (for example categories, series) for categorize content, Vocabularies - where are terms are grouped and tags.

Create plugin bin/cake bake plugin CakeTaxonomy.

Create tables

bin/cake bake migration Vocabularies name:string description:text created modified -p CakeTaxonomy
bin/cake bake migration Terms name:string description:text slug:string vocabulary_id:integer created modified -p CakeTaxonomy
bin/cake bake migration Tags name:string description:text slug:string created modified -p CakeTaxonomy

run

bin/cake migrations migrate -p CakeTaxonomy

and build all items for newly created tables

bin/cake bake all Vocabularies -p CakeTaxonomy
bin/cake bake all Terms -p CakeTaxonomy
bin/cake bake all Tags -p CakeTaxonomy

Slugs

If you want to use multi-word terms in html it will be good to create slugs so install utility package composer require jdmaymeow/cake-utility. and update your terms table model. Add intio initialize function

$this->addBehavior('CakeUtility.Sluggable', [
    'field' => 'name'
]);
